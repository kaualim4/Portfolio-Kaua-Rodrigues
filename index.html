MINHAS ESPECIALIDADES.

HTML

É uma linguagem de marcação usada para organizar a estrutura e o conteúdo das páginas da internet, especificando elementos como texto, imagens, links e formulários através de etiquetas.

CSS

É uma linguagem de estilo que controla a apresentação visual das páginas web, permitindo definir o layout, as cores, as fontes e outros aspectos visuais, separando a apresentação do conteúdo HTML.

JavaScript

É uma linguagem de programação de alto nível que possibilita a interatividade e a dinamicidade nas páginas web. Usado para manipular elementos HTML, responder a eventos do usuário, validar formulários, criar animações e muito mais.
